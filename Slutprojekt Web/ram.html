Några av de största tillverkarna av RAM minnen är: Kingston, Corsair, HyperX och G.Skill.

RAM, eller Random Access Memory är som vårat korttidsminne. När man använder datorn lagras saker här för att snabbare kunna komma åt igen. I datorn finns även ett långtidsminne, det kallas HDD eller SSD och används för generellt lagrande.
